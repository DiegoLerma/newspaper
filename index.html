Bajo Reserva Exprés: Alumno de la IEU ahí la lleva para aprender HTML

Segun testigos, el estudiante guanajuatense se pasó toda la tarde viendo cursos y leyendo los apuntes digitales para rifarse escribiendo este artículo

Guanajuato, Guanajuato, México, 02 de noviembre de 2020

El estudiante guanajuatense Diego Lerma, con el apoyo del Dr. Alfonso Medel Rojas y los apuntes digitales, después de toda la semana de práctica y lectura de principios básicos de HTML, escribió su primer artículo para navegador web para la clase de Programación Web del Instituto de Estudios Universitarios de Puebla.

El artículo presentado por el estudiante capitalino todavía no está disponible en los medios oficiales de difusión de noticias especializadas, como El Universal o El Excelsior.

Actualmente, el artículo ya fue presentado a sus profesores para su revisión y que estos otorguen el veredicto final sobre la calidad y correcciones en el primer código realizado.

El alumno se muestra deseoso de aprender más acerca de HTML, CSS3 y JS dentro de poco, para posteriormente incursionar en el ámbito de React Native o Electron
